# Примеры организации изображений

## Рекомендуемая структура для images/

```
images/
├── hero/              # Изображения для hero-секций
│   ├── hero-main.jpg
│   └── hero-services.jpg
├── services/         # Изображения услуг
│   ├── expertise/
│   ├── evaluation/
│   └── laboratory/
├── blog/             # Изображения для блога
│   └── posts/
├── about/            # Изображения страницы "О компании"
│   └── team/
└── common/           # Общие изображения
    └── placeholders/
```

## Примеры именования

- `hero-main.jpg` - главное hero изображение
- `service-expertise-auto.jpg` - услуга автоэкспертизы
- `blog-post-1.jpg` - изображение для статьи блога
- `about-office-spb.jpg` - офис в СПб
- `team-expert-ivanov.jpg` - фото эксперта
