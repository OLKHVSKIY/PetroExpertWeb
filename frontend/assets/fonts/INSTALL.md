# Установка шрифта Onset

## Шаг 1: Получите файлы шрифта

Вам нужно получить следующие файлы шрифта Onset:
- `Onset-Regular.woff2` (рекомендуется)
- `Onset-Regular.woff` (опционально)
- `Onset-Regular.ttf` (опционально)
- `Onset-Bold.woff2` (рекомендуется)
- `Onset-Bold.woff` (опционально)
- `Onset-Bold.ttf` (опционально)

## Шаг 2: Поместите файлы в папку

Скопируйте файлы шрифта в папку:
```
frontend/assets/fonts/
```

## Шаг 3: Проверка

После добавления файлов, обновите страницу в браузере. Шрифт Onset должен автоматически загрузиться.

Если файлы не найдены, браузер будет использовать шрифт Inter как fallback.

## Альтернативный способ

Если у вас есть только TTF файлы, вы можете конвертировать их в WOFF2 используя онлайн-конвертеры:
- https://cloudconvert.com/ttf-to-woff2
- https://convertio.co/ttf-woff2/

## Текущий статус

Код подключения шрифта активирован в `css/main.css`. Файлы шрифта ожидаются в папке `assets/fonts/`.
